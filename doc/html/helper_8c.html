◆ parseTimeSpan()

int parseTimeSpan ( char *  str)

Check the string for a timespan format, return a minute integer.

  • an integer and a legal type (example: 1min, 2hour, 1day)
  • a floting point number and a legal type (example: 2.5min 1,2h, 0.03d)
  • just an integer is parsed as minutes
  • legal types:
    • min / m / minute
    • hour / h
    • day / d multiply the number with the correct multiplier to return time in minutes. use: <multiplierForType>"("char* type")"
Parameters
[in]strthe time span string, number and type without space
Return values
positiveinteger the time in minutes on SUCCESS
-1FAILURE
